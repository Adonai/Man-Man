Man-Man (GPLv3)
=======

Метка "Покупки в приложении" относится только к кнопке "Помочь проекту", никаких закрытых возможностей в программе не имеется

Удобное приложение по поиску, объяснению и просмотру мануалов команд Linux
Клиент для андроида, использующий API и содержимое сайта www.mankier.com

Это утилита для любого смышлёного в Linux человека, знакомого с принципом ман-страниц. Предоставляет удобный способ быстрой навигации, просмотра и сохранения мануалов на ваш телефон или планшет.

Возможности:
--------

- Поддерживает поиск одиночной команды
- Поддерживает объяснение простых однострочников
- Поддерживает просмотр и индексацию глав ман-страниц
- Поддерживает кэширование просмотренных ранее мануалов

В работе:
-----------

- Загрузка и просмотр TROFF-файлов из локального архива мануалов (*.gz)
- Действия для страниц (загрузка всех результатов поиска в кэш, очистка по регекспам, избранное и т.д.)
